## esvtoleet

The ESV Bible passage morse translator

#### How to use

```bash
$ python esv-to-morse.py passage key
```

or you can run the script and then enter the passage

```bash
$ python esv-to-morse.py
```


| parameter | description                             | comment |
| :---------| :-------------------------------------- | :------ |
| passage   | ex "John 3:16"                          | optional |
| key       | if you have a key at the ESV Bible Web Service API (http://www.esvapi.org/v2) | optional |
